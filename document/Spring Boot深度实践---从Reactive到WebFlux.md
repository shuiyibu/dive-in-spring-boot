# 理解Reactive

一些说法：

- Reactive是异步非阻塞编程
- Reactive能够提升程序性能
- Reactive能够解决传统编程模型遇到的困境





- 反应堆模式（Reactor）----同步非阻塞（多工）

- Proactor模式-------异步非阻塞（多工）
- 观察者模式（Observer）---事件监听（推push的模式：数据由客户端推送到服务端）
- 迭代模式---（拉pull的模式）
- Java并发模型
